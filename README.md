# Online Examination System

## 📌 Project Overview
The **Online Examination System** is a web-based application developed to conduct online examinations digitally.  
The system allows students to register, log in, attend exams, submit answers, and view results automatically.  
The admin can manage students, exams, questions, and results efficiently.

## 🎯 Project Objectives

- Conduct examinations online
- Reduce manual paperwork
- Generate results automatically
- Manage exam and student records digitally
- Improve speed and accuracy in the examination process

## 👥 Users of the System

### Admin
- Admin Login
- Add Students
- Create Exams
- Add Questions
- View Results

### Student
- Registration/Login
- Start Exam
- Submit Answers
- View Result
